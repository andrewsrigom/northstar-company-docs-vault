# AI Tools Working Group Notes

Status: working notes
Owner: IT Security and Legal
Last reviewed: 2026-04-10

## Context

These notes capture unresolved discussion about employee use of AI tools. They are not a final policy.

## Questions Under Review

- approved AI tool list;
- retention and training settings;
- whether source code can be used for debugging;
- customer data redaction requirements;
- audit logging expectations;
- procurement review for AI vendors;
- employee training requirements.

## Current Guidance

Until a final policy exists, employees should follow the draft AI tool usage page and avoid using Confidential or Restricted data in unapproved tools.
