# Phishing Reporting

Owner: IT Security
Last reviewed: 2026-01-27

## Purpose

This runbook explains how employees report suspected phishing and what IT Security does after a report.

## Employee Steps

If a message looks suspicious, use the Report Phishing button when available. If the button is unavailable, forward the message as an attachment to security@northstar.example and open an IT Support ticket.

Do not click links, open attachments, reply to the sender, or enter credentials after reporting.

## If You Clicked

If you clicked a link, opened an attachment, approved an MFA prompt, or entered credentials, mark the ticket as urgent and post in the Security Hotline channel. IT Security may reset passwords, revoke sessions, or isolate the device.

## Security Triage

IT Security reviews sender, links, attachments, authentication results, and similar messages received by other employees. Confirmed campaigns may trigger mailbox search, block rules, and awareness messaging.
