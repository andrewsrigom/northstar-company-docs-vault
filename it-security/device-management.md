# Device Management

Owner: IT Security
Last reviewed: 2026-02-28

## Purpose

This policy explains how company-managed laptops and mobile devices are issued, secured, supported, and returned.

## Standard Controls

Company laptops must have disk encryption, endpoint protection, automatic screen lock, operating system updates, and device management installed. Employees should not disable security tools or share local accounts.

## Personal Devices

Personal devices may access approved cloud applications only when MFA is enabled and the device meets baseline browser and operating system requirements. Customer data should not be downloaded to unmanaged personal devices.

## Repairs and Replacement

Employees should open an IT Support ticket for hardware issues. IT may ship a replacement device when repairs would block work for more than two business days.

## Return

Departing employees must return company devices using the shipping label provided by People Operations or IT. Managers should not accept returned laptops directly unless IT has approved the process.
