# Password Manager Guide

Owner: IT Security
Last reviewed: 2026-02-02

## Purpose

Northstar uses a company password manager for shared credentials, emergency access records, and approved secrets that cannot yet be replaced by SSO or managed vaults.

## Employee Use

Employees should store unique passwords in the company password manager. Passwords should not be reused across personal and company accounts.

## Shared Vaults

Shared vaults require a system owner, access group, and quarterly review. Do not share passwords in Slack, tickets, documents, screenshots, or calendar invites.

## Emergency Access

Emergency access records are limited to business-critical systems and must have at least two approved recovery owners. IT Security reviews emergency access quarterly.

## Credential Rotation

Rotate credentials after employee offboarding from shared access, suspected compromise, vendor team changes, or system ownership changes.
