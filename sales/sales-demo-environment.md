# Sales Demo Environment

Owner: Sales Engineering
Last reviewed: 2026-02-25

## Purpose

The demo environment lets Sales show realistic workflows without exposing customer data.

## Rules

Use only fictional accounts, fictional users, synthetic documents, and approved demo scenarios. Do not import customer exports, screenshots, private notes, or production credentials.

## Requests

Sales can request demo data changes through Sales Engineering. Requests should include scenario, persona, desired workflow, deadline, and whether the demo is for a strategic opportunity.

## Incidents

If customer data is accidentally added to the demo environment, report it as a security incident and notify Sales Engineering.
