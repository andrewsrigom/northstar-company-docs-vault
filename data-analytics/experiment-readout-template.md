# Experiment Readout Template

Owner: Data Analytics and Product
Last reviewed: 2026-02-28

## Purpose

Use this template to summarize product, marketing, onboarding, or support workflow experiments.

## Template

- experiment name;
- owner;
- hypothesis;
- audience;
- start and end date;
- success metric;
- guardrail metrics;
- result;
- decision;
- follow-up owner.

## Interpretation

Do not treat directional results as conclusive when sample size is small, tracking changed mid-experiment, or user segments were mixed.

## External Claims

Experiment results should not be used in external marketing claims without Marketing, Legal, and Data Analytics review.
