# Data Access Review

Owner: Data Analytics and IT Security
Last reviewed: 2026-03-16

## Purpose

This page explains how employees request access to analytics data, BI tools, and data exports.

## Access Requirements

Requests must include business purpose, data needed, duration, manager approval, and whether customer or employee data is included.

## Restricted Data

Payroll, employee records, security logs, legal records, secrets, and raw customer content require additional owner approval and may not be available in standard analytics tools.

## Review

Data Analytics confirms data source and role. IT Security reviews sensitive access. Legal reviews privacy concerns when personal data is used in a new way.

## Expiration

Temporary access should expire automatically. Long-lived access is reviewed quarterly.
